Admin Access

To access the admin dashboard, go to the Admin page in the navigation bar. The password is commack2026. From there you can approve or reject submitted items, review claims, and view transaction history.

Technologies Used

This website was built from scratch using Python with the Flask web framework. Data is stored in a SQLite database through SQLAlchemy. The front end is plain HTML, CSS, and JavaScript with no templates or frameworks. All code is original.
