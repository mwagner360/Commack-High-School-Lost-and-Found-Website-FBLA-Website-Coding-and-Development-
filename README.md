# Commack High School Lost and Found Website

FBLA Website Coding & Development  
Developed by Michael Wagner

---

## Project Overview

This project is a Lost and Found website created for Commack High School. The goal of the website is to make it easier for students and staff to report and find lost items in one central place instead of relying on announcements or word of mouth.

The site allows users to view items that have been found, submit new items, and contact the finder if they believe an item belongs to them. This project was created for the FBLA Website Coding & Development competitive event.

---

## Problem

Lost items are common in schools, but there is no organized system for keeping track of them. Items often end up in offices or classrooms with no easy way for students to know what has been found.

This project solves that problem by providing an online platform where found items can be posted and searched.

---

## What the Website Does

The website allows users to:
- View a list of found items
- Submit a found item with basic details
- Search and browse items
- Submit a claim or inquiry for an item
- Manage listings through an admin interface

The design makes it accessible for anyone can navigate the site without specific instructions.

---

## Technologies Used

- Python with Flask
- SQLAlchemy for the database
- HTML and CSS for the frontend
- Replit for hosting and deployment

---

## IMPORTANT NOTE FOR JUDGES

In some cases, an "Internal Server Error" may appear when clicking buttons or moving between pages.

This is not a coding or logic issue with the project. It happens occasionally due to how the application is deployed on Replit.

If this happens, **just refresh the page**. After refreshing, the website works normally. (Sometimes more than one refresh click is needed)

This issue:
- Does not affect saved data
- Does not break functionality
- Is related to deployment, not the code itself

## Running the Project Locally

To run the project locally:

1. Clone the repository:
   git clone https://github.com/mwagner360/Commack-High-School-Lost-and-Found-Website-FBLA-Website-Coding-and-Development-

2. Go into the project folder:
   cd Commack-High-School-Lost-and-Found-Website-FBLA-Website-Coding-and-Development-

3. Install dependencies:
   pip install -r requirements.txt

4. Run the application:
   python app.py

5. Open a browser and go to:
   http://localhost:5000
