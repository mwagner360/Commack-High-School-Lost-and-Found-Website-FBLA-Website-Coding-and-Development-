Accessibility

We built this site to be usable by everyone at Commack High School, including people who rely on screen readers, keyboard navigation, or other assistive tools.

Features

The site uses semantic HTML so screen readers can understand the page layout. All form inputs have labels and images have alt text where it matters. There's a skip to content link at the top of every page for keyboard users and all interactive elements have visible focus indicators.

We picked colors that are easy to read against all backgrounds, and the layout is responsive so it works on phones, tablets, and desktops.

Issues

If something on this site doesn't work for you, let us know and we will fix it.
